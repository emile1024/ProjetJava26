# ProjetJava26

## Auteur
Emile Nibasumba et Josue Mulumba Mutanda

## Description
Programme Java qui gère les étudiants, leurs cours et leurs notes.

## Fonctionnalités
- Ajouter/modifier une note
- Supprimer un étudiant
- Afficher tous les étudiants
- Afficher étudiants incomplets
- Trier étudiants par moyenne 
- Exporter en CSV

## Exécution
Lancer la classe Main.java
